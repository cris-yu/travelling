# 夕阳红旅游应用 - 项目特性总结

## 📱 应用概述

这是一款专为50-70岁中老年用户设计的跟团游/一日游移动应用，采用React + Vite技术栈开发，注重用户体验和无障碍设计。

## 🎨 UI/UX 设计亮点

### 字体大小体系
```
超大: 48px - 集合时间（最重要）
特大: 28-32px - 价格、重要数字
大: 20-24px - 标题
中: 17-19px - 按钮文字、重要内容
标准: 16px - 正文基础字号
小: 14-15px - 辅助信息
```

### 颜色系统
```
主色: #1677ff - 品牌蓝，信任感
危险: #ff4d4f - 紧急、重要信息
成功: #52c41a - 医疗、确认
警告: #faad14 - 提示、注意
文字: #333 - 高对比度正文
辅助: #666 - 次要信息
背景: #f5f5f5 - 柔和背景
```

- ✅ 最小点击区域44x44px

- ✅ 平滑动画过渡
- ✅ 清晰的视觉反馈
