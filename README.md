# 老友行 - 中老年专属跟团游小程序

## 项目简介

老友行是一款专为50-70岁中老年用户设计的跟团游/一日游微信小程序。项目专注于中老年用户体验优化，提供大字体、简化操作、一键功能等人性化设计。

## 功能特性

### 核心功能
- **产品筛选与展示**：京津冀地区旅游产品，标注50-70岁适龄
- **产品详情**：体力强度、医疗保障、交通安排详细说明
- **预订流程**：支持子女代订、年龄验证、分享功能
- **行程管理**：我的行程列表，订单信息查看
- **行程详情**：集合时间超大字体显示、一键导航、一键拨打

### 中老年用户优化
- **大字体设计**：基础字体32rpx，关键信息最大96rpx
- **高对比度配色**：重要信息使用红色突出显示
- **大按钮设计**：所有按钮最小高度88rpx
- **简化操作**：移除底部TabBar，使用页面内导航
- **一键功能**：一键拨打电话、一键导航到集合地点
- **全局客服**：悬浮客服按钮，紧急求助功能

## 技术栈

- **框架**：微信小程序原生开发
- **语言**：JavaScript, WXML, WXSS
- **API**：微信小程序官方API
  - `wx.makePhoneCall()` - 拨打电话
  - `wx.openLocation()` - 地图导航
  - `wx.navigateTo()` - 页面跳转
  - `wx.setClipboardData()` - 复制信息
  - `wx.showToast()` / `wx.showModal()` - 提示框

## 快速开始

### 环境要求
- 微信开发者工具（最新版本）
- 微信小程序账号（可使用测试号）

### 安装步骤

1. **克隆项目**
```bash
git clone https://github.com/cris-yu/travelling.git
cd travelling
```

2. **打开项目**
   - 启动微信开发者工具
   - 点击"导入项目"
   - 选择项目目录：`/path/to/travelling`
   - AppID：选择"测试号"或填入你的AppID
   - 项目名称：老友行

3. **运行项目**
   - 导入成功后，项目会自动编译运行
   - 在模拟器中即可查看效果

## 主要页面说明

### 1. 首页（index）
- 搜索框：支持搜索目的地或产品名称
- 筛选功能：按目的地、体力强度筛选
- 产品列表：展示旅游产品卡片

### 2. 产品详情页（detail）
- 产品完整信息展示
- 体力强度、医疗保障、交通安排
- 详细行程安排
- 预订按钮

### 3. 预订页（booking）
- 选择出行日期
- 添加多位出行人信息
- 身份证号验证及年龄检查
- 联系人信息填写
- 订单提交

### 4. 我的行程页（orders）
- 订单列表展示
- 关键信息突出：集合时间、集合地点、导游电话
- 一键拨打、一键导航
- 分享行程给家人

### 5. 行程详情页（itinerary）
- 超大字体显示集合时间（96rpx）
- 当日行程时间轴
- 紧急联系功能
- 一键导航到集合地点

## 设计亮点

### 针对中老年用户的贴心设计

1. **超大字体显示集合时间**
   - 使用96rpx字体
   - 红色高亮显示
   - 避免老年人看错时间

2. **一键拨打功能**
   - 点击电话号码直接拨打
   - 无需手动输入

3. **一键导航功能**
   - 点击地址直接打开地图导航
   - 无需复制地址

4. **紧急求助**
   - 全局悬浮客服按钮
   - 紧急求助快速通道

5. **分享功能**
   - 一键分享行程给家人
   - 子女可随时查看父母行程

## 数据说明

当前使用模拟数据（`utils/data.js`），包含4个示例产品：
- 北京故宫一日游
- 天津古文化街一日游
- 承德避暑山庄两日游
- 秦皇岛北戴河休闲三日游

实际使用时，可替换为后端API接口数据。

## 图片说明

产品图片使用base64编码的SVG占位符：
- 北京故宫：红色 #FF6B6B
- 天津古文化街：青色 #4ECDC4
- 承德避暑山庄：绿色 #95E1D3
- 秦皇岛北戴河：粉色 #F38181

正式使用时，请替换为真实产品图片，并在小程序后台配置图片服务器域名。
