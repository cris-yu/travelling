# 夕阳红旅游 - 中老年专属跟团游应用

## 项目简介

这是一款专为50-70岁中老年用户设计的跟团游/一日游移动应用。应用采用大字体、高对比度、简化操作等设计原则，确保中老年用户能够轻松使用。

## 核心功能

### 1. 产品筛选与展示
- 精选适合中老年用户的旅游目的地
- 仅包含京津冀地区产品
- 明确标注适合年龄段（50-70岁）
- 支持按目的地、体力强度筛选

### 2. 产品详情页
- 清晰展示体力强度等级
- 医疗保障说明（医护人员、急救措施）
- 交通安排详情（车型、座位、设施）
- 详细行程时间轴

### 3. 预订流程
- 简单直观的下单操作
- 支持子女代为预订
- 自动年龄验证
- 订单分享功能

### 4. 行程展示
- 当日关键信息醒目显示
  - 集合时间（超大字体）
  - 集合地点（一键导航）
  - 导游电话（一键拨打）
- 详细行程时间轴

### 5. 客服与应急
- 全局悬浮客服按钮
- 一键联系客服
- 紧急求助热线
- 24小时服务保障

## 技术栈

- **前端框架**: React 18
- **构建工具**: Vite 5
- **路由**: React Router 6
- **UI组件**: Ant Design Mobile 5
- **样式**: CSS3

## 安装与运行

### 前提条件
- Node.js 16+ 
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

应用将在 http://localhost:3000 启动

### 构建生产版本
```bash
npm run build
```

### 预览生产构建
```bash
npm run preview
```

## 核心设计理念

### 针对中老年用户的优化

1. **大字体设计**
   - 基础字号16px
   - 关键信息最大48px（集合时间）
   - 保证清晰易读

2. **高对比度**
   - 文字与背景高对比
   - 重要信息使用醒目颜色
   - 避免低对比度组合

3. **简化操作**
   - 减少操作步骤
   - 大按钮设计（最小44x44px）
   - 一键功能（拨打、导航）

4. **信息层级明确**
   - 关键信息突出显示
   - 使用视觉引导
   - 避免信息过载

5. **友好的错误处理**
   - 清晰的提示信息
   - 年龄验证友好提醒
   - 表单验证实时反馈
