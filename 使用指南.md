# 项目使用指南

## 快速开始

由于系统中没有安装Node.js，您需要先安装Node.js环境才能运行此项目。

### 步骤1: 安装Node.js

1. 访问 https://nodejs.org/
2. 下载并安装LTS版本（推荐）
3. 验证安装：打开终端运行
   ```bash
   node --version
   npm --version
   ```

### 步骤2: 安装项目依赖

在项目根目录下运行：
```bash
npm install
```

这将安装所有必要的依赖包，包括：
- React 18
- React Router 6
- Ant Design Mobile 5
- Vite 5

### 步骤3: 启动开发服务器

```bash
npm run dev
```

应用将在 http://localhost:3000 启动并自动在浏览器中打开。

## 项目页面导航

### 1. 首页 - 产品列表 (/)
- 展示所有京津冀地区的旅游产品
- 支持按目的地和体力强度筛选
- 搜索功能
- 点击产品卡片查看详情

### 2. 产品详情页 (/product/:id)
- 查看产品完整信息
- 体力强度、年龄范围、医疗保障
- 交通安排、行程亮点
- 详细行程时间轴
- 点击"立即预订"进入预订页

### 3. 预订页面 (/booking/:id)
- 选择出行日期
- 填写出行人信息（支持子女代订）
- 填写联系人信息
- 自动计算订单总价
- 提交订单

### 4. 我的行程 (/my-orders)
- 查看已预订的行程
- 关键信息突出显示（集合时间、地点、导游电话）
- 一键拨打、一键导航功能
- 分享行程给家人
- 查看详细行程

### 5. 行程详情页 (/itinerary/:orderId)
- 当日行程醒目展示
- 集合时间（超大字体48px）
- 一键导航到集合地点
- 一键拨打导游电话
- 完整行程时间轴
- 温馨提示
- 紧急联系方式

## 注意事项

1. **模拟数据**：当前使用模拟数据，实际项目需要接入真实API

2. **地图功能**：导航功能会尝试打开地图应用，需要在真实设备上测试

3. **电话功能**：拨打电话功能在浏览器中会打开拨号界面

4. **分享功能**：使用Web Share API，在不支持的浏览器中会降级为复制到剪贴板

5. **移动端优化**：本应用专为移动端设计，建议使用移动设备或浏览器的移动设备模拟器查看

## 常见问题

**Q: 如何在手机上预览？**
A: 
1. 启动开发服务器后，查看终端显示的局域网地址
2. 确保手机和电脑在同一WiFi网络
3. 在手机浏览器中访问该地址

**Q: 为什么导航和拨打电话功能没反应？**
A: 这些功能需要在真实移动设备上测试，浏览器桌面版可能不支持

**Q: 如何添加更多产品？**
A: 编辑 `src/data/mockData.js` 文件，在 `mockProducts` 数组中添加新产品

**Q: 如何修改客服电话？**
A: 编辑以下文件中的电话号码：
- `src/components/CustomerService.jsx`
- `src/pages/ItineraryDetail.jsx`
